pub fn as_int_lit(self) -> Option<IntLiteral<'a>>

If self is one of the integer literals, returns it in a nicer-to-work-with form.

Unfortunately Token has to work like this to fit in 24 bytes, if it just held IntLiteral in the variants it would be 32.

source

pub fn as_float_lit(self) -> Option<FloatLiteral<'a>>

If self is one of the float literals, returns it in a nicer-to-work-with form.

Unfortunately Token has to work like this to fit in 24 bytes, if it just held FloatLiteral in the variants it would be 32.
