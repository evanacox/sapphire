Expand description

Defines the API for a machine instruction.

Each specific target has a unique set of possible MachInsts (this is what makes up the unique “MIR” (machine instruction representation) for that target), but they all provide the same basic operations. This enables cross-target code sharing for things like instruction scheduling and register allocation.

fn uses<const N: usize>( &self, frame: &dyn StackFrame<Self::Arch>, collector: &mut RegCollector<N> )

Pushes every register being used by self into collector.

It can be assumed that if the live ranges of these end at this instruction that they can be used as the destination.
