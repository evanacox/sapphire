Module sapphire::codegen

source ·
Expand description

APIs for the compiler back-end and code-generation facilities

These APIs mostly abstract away any target-specific details, but there is target-specific code in this module. Anything in a module with an architecture in its name (e.g. sapphire::codegen::x86_64::* or sapphire::codegen::aarch64::*) are CPU-specific.
