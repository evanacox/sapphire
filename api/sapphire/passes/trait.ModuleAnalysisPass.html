fn expects_preserved(&self) -> &'static [TypeId]

Allows analyses to declare dependence on other analyses.

This directly implies that if T is preserved, all analyses referred to by T::expects_preserved are also assumed to be preserved.

This is checked in debug mode.

source

fn run(&mut self, module: &Module, am: &ModuleAnalysisManager) -> Self::Result

Performs the analysis and returns a computed result. This should not be an impure operation, running the analysis twice on the same input should produce the same result.
