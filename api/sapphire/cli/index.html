Module sapphire::cli

source ·
Expand description

Contains utility code specifically for the CLI tools located in the tools/ subdirectory.

All of these tools have similar command-line arguments and they all should look/feel uniform, so most of the code is pulled into this module and then used in the drivers of the different tools.
