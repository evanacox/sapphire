Struct sapphire::analysis::ModuleWriter

source · 
pub struct ModuleWriter { /* private fields */ }
Expand description

A simple SIR -> text pass that takes in an entire module, turns it into textual SIR, and then maps each IR entity to a range of text referring to it.

This can be used for debug/test passes that need to produce human-readable SIR.
