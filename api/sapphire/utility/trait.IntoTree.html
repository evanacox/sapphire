Expand description

Models a type that can be traversed in a tree-like fashion. This is intended for debug APIs / passes that deal in human-readable trees.

This is not for anything that is actually in the hot path of any compiler passes, this is meant for human-facing APIs. What this means is that it may be extremely slow.
