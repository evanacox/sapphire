Struct sapphire::ir::Layout

pub struct Layout { /* private fields */ }
Expand description

Models the layout of an entire function and every basic-block in it.

Each block is modeled as a linked list to allow easy splicing and in-place editing, and the list of blocks is also modeled as a linked list for similar reasons.

pub fn loc_compare(&self, a: Inst, b: Inst) -> Ordering

Compares two instructions that are from the same block, and returns their relative order compared to each-other.

Ordering::Less => a < b order-wise, i.e. b comes after a. Ordering::Equal => a == b. Ordering::Greater => a > b order-wise, i.e. a comes after b.
