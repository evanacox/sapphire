Expand description

Some instructions model unary operations, those instructions model this trait.

A valid assumption for any type implementing this trait is that the operands returned by Instruction::operands has a length of exactly 1.

While most instructions implementing this are casts, not all of them are. A few instructions (notably fneg) are unary but not casts.
