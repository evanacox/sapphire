Expand description

Helper trait that allows easy creation of instruction builders. This trait provides a variety of helper methods that build instructions and inserts them in whatever way the trait implementor defines.

This is used for the append/insert/replace builders, along with any other more situational ones scattered around the codebase.

Gets the data-flow graph in use for the inserter

source

fn build(self, data: InstData, debug: DebugInfo) -> (Inst, Option<Value>)

“Builds” a single instruction and inserts it in whatever way the particular InstBuilder sees fit.

This returns a reference to the instruction and possibly a reference to the result of that instruction.
