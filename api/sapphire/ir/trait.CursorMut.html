source

fn replace(&mut self) -> ReplaceBuilder<'_>

Returns an instruction builder that replaces the current instruction.

source

fn insert(&mut self) -> InsertBuilder<'_>

Allows an instruction to be inserted before the current one being pointed at.

If this points at At(inst), an instruction is inserted before inst. If this is pointing at After(bb), an instruction is appended. Either way, the cursor is not moved, so repeated Self::insert calls make the instructions appear in the same order in the program.

source

fn replace_uses_with(&mut self, original: Value, new: Value)

Replaces any uses of original in the function with new.

source

fn remove_inst(&mut self) -> Inst

Removes the current instruction and leaves the cursor pointing at the instruction immediately following the one that was removed.

Returns the instruction that was removed.

source

Removes the current instruction and leaves the cursor pointing at the instruction immediately before the one that was removed.

Returns the instruction that was removed.

fn create_block(&mut self, name: &str) -> Block

Creates a single basic block and returns it. This block is appended to the end of the block list.

Note that this does not switch the cursor to operate on that block, you still need to call goto_before or similar.

source

fn create_block_before(&mut self, name: &str, before: Block) -> Block

Equivalent to Self::create_block, except it inserts the block before before instead of appending it.

source

fn create_block_after(&mut self, name: &str, after: Block) -> Block

Equivalent to Self::create_block, except it inserts the block after after instead of appending it.

source

fn rewrite_branch_target( &mut self, to: Block, new: BlockWithParams ) -> BlockWithParams

Rewrites the branch at the end of the current block that targets to to instead branch to new (with the args associated with new).

Returns the old branch target so you can re-use the arguments.
