Expand description

Models basic cursor operations that view a function. None of these operations require mutable access to a given function, so they can be used inside of analyses.

fn next_block(&mut self) -> Option<Block>

Moves the cursor to the next block in the function. If the cursor is currently not pointing to anywhere in the function, this moves it to Before(entry). If the cursor is pointing at the last block in the function, this moves it to Nothing.

source

fn prev_block(&mut self) -> Option<Block>

Moves the cursor to the block before the current one. If the cursor is pointing at Nothing, nothing changes. If the cursor is pointing at the first block, its moved to Nothing.

source

fn next_inst(&mut self) -> Option<Inst>

Moves the cursor to the next instruction in the function. If the cursor points before the block, this is the first instruction. If it points after, this does nothing. If it points at nothing, this does nothing.

source

fn prev_inst(&mut self) -> Option<Inst>

Moves the cursor to the previous instruction in the function. If the cursor points after the block, this is the last instruction. If it points before, this does nothing. If it points at nothing, this does nothing.
