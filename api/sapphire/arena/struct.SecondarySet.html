Struct sapphire::arena::SecondarySet

source · 
pub struct SecondarySet<K: ArenaKey> { /* private fields */ }
Expand description

Intended to be a dense representation for a set of keys from a primary map.

This is theoretically equivalent to a SecondaryMap<K, ()> but is a bit leaner under-the-hood, being implemented as just a bitvector with no additional slot storage.

arena_key! { struct Key; }
let mut map = ArenaMap::default();
let k1: Key = map.insert(15);
let k2 = map.insert(20);
let secondary = SecondarySet::map_keys(&map, |m, k| m[k] % 2 == 0);

assert_eq!(secondary.contains(k1), false);
assert_eq!(secondary.contains(k2), true);

pub fn map_keys<T, F>(primary: &ArenaMap<K, T>, f: F) -> Selfwhere F: FnMut(&ArenaMap<K, T>, K) -> bool,

Efficiently creates a SecondarySet by applying a function to each key inside of primary to determine if a key should be included in the set or not.

Effectively, this results in the set $$ { k | f(k) is true } $$

arena_key! { struct Key; }
let mut primary = ArenaMap::<Key, i32>::new();
let k1 = primary.insert(4);
let k2 = primary.insert(5);

let is_even = SecondarySet::map_keys(&primary, |map, key| map[key] % 2 == 0);

assert_eq!(is_even.contains(k1), true);
assert_eq!(is_even.contains(k2), false);

pub fn keys(&self) -> impl Iterator<Item = K> + '_

Returns an iterator that iterates over the keys that are present in the set. Keys are guaranteed to be yielded in increasing order, in the range $$ [0, C) $$ such that $$ C $$ is cardinality().

let secondary = SecondarySet::map_keys(&map, |m, k| m[k] % 2 != 0);
let mut it = secondary.keys();
assert_eq!(it.next(), Some(k1));
assert_eq!(it.next(), None);
